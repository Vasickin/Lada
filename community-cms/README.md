# Система управления проектами организации "ЛАДА"

## Описание проекта

Система управления проектами для русской общины "ЛАДА". Включает управление проектами, командой, партнерами, видео и фотогалереями.

## Технологии

- **Java 17**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **Spring Security** (интеграция с существующей системой)
- **MySQL 8.0**
- **Flyway** (миграции базы данных)
- **Thymeleaf** (шаблоны)
- **Bootstrap 5** (интерфейс)
- **OpenAPI 3** (документация API)
- **Docker & Docker Compose** (контейнеризация)
